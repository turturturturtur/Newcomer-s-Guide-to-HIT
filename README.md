本人受到《上交大生存指南》《cs自学指南》的感召，决定创作这样一个平台，来帮助各位工大学子。

本书希望提供人生若干道路的建议，整合工大各种资源和社会资源，成为工大学子的参考平台，也帮助各位尽快适应大学生活，走进自己的节奏。

本书希望做一个灯塔，照亮学子人生启航的道路。

> **从今往后，你是你自己人生的掌舵人。**
